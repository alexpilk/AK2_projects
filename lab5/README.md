# Integrals
Clean: `make clean`  
Build: `make`  
Run: `make run`  
Clean, build and run: `make all`
Debug: `make debug`

## Example
1. Start debugging `make debug`
2. Press `<Enter>`
3. Set breakpoint at exit e.g. `b 53`
4. Run `r`
5. Check number in memory `i fl`
